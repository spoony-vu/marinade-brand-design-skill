Setup
How to Install
1

Add the skill to Claude Code

Use the buttons above to download SKILL.md or run the npx command. Place SKILL.md and assets/ into ~/.claude/skills/marinade-brand-design/

2

Start prompting

The skill auto-activates when it detects Marinade presentation work. Works with any format — HTML, React, Marp, PPTX, or video:

# In Claude Code, try: "Create a TAM/SAM/SOM slide for Marinade. TAM: $82B. SAM: $17B. SOM: $1.5B." # Or a full deck: "Build a 6-slide deck for Marinade with a staking flow diagram, competitive comparison table, and a TVL growth bar chart."
3

Brand consistency enforced automatically

Every slide maps to one of 16 master archetypes. The skill applies exact colors, typography, grid spacing, and tone of voice. Diagrams use the brand teal spectrum. WCAG contrast ratios are pre-computed for every color pair.

opt

Download the master slide library (optional)

For access to all source images, logos, and slide layouts, download the master template as a .pptx and point Claude to it:

# Download from Google Slides (File → Download → .pptx) docs.google.com/presentation/.../Marinade Master Slide Template

The skill works without this — brand rules are built in. The .pptx gives Claude access to original imagery when extracting assets.

Color Accessibility

Pre-computed contrast ratios for every foreground/background pair in the brand palette.

ForegroundBackgroundRatioAAAAANotes
Dark PrimaryWhite17.6:1YesYesDefault body text
Deep TealWhite10.6:1YesYesHeadings on light
Brand TealWhite4.0:118pt+NoLarge text only
Muted TextWhite4.0:118pt+NoCaptions, labels only
White#19454410.6:1YesYesText on teal slides
Light Teal#151A1A9.6:1YesYesAccent on dark
White#308D8A4.0:118pt+NoLabels on teal fills

Body text on white

Use Dark Primary, Deep Teal, or Body Secondary only. Brand Teal and Muted Text fail AA at body sizes.

Brand Teal is 18pt+ only

4.0:1 ratio — safe for metric numbers (36pt+), headings, and labels. Never for body paragraphs.

Icons need context

Meaningful icons: add aria-label. Decorative icons paired with text: aria-hidden="true". Min 44px touch target.

Rules
Key Design Constraints

These rules keep every slide consistent with the master template.

Light teal default

Default background is #EDF5F3. Deep teal gradient and teal gradient for separator/accent slides only.

Shadows & gradients

Subtle box-shadows allowed for depth (teal-tinted, low opacity). Gradients within the teal spectrum. Eased gradients preferred over linear.

60-70% fill maximum

Never fill more than 70% of a slide with content. White space is structural and intentional.

Sentence case only

Never ALL CAPS except acronyms (TAM, SAM, SOM, APY, SOL). No exclamation marks.

PT Serif is rare

Maximum one italic accent word per slide. Only in title compositions contrasting with DM Sans. Never for body text.

Teal only accents

All accent work stays within the teal spectrum. Yellow (#FEF29D) reserved for Marinade's position marker only.

Voice
Tone of Voice

Concise, intelligent, calm confidence. Metrics over adjectives.

Use
"8.81% APY through automated delegation across 400+ validators"
"3 reasons why NOT to liquid stake your Solana"
"$1.5B addressable market in Solana liquid staking"
Avoid
"Revolutionary industry-leading yield optimization platform"
"Unlock the full potential of your staked assets today!"
"Disrupting the staking landscape with next-gen technology"

Mermaid diagrams

Uses beautiful-mermaid color-mix() pattern — all diagram colors derive from two base variables.

/* Marinade Mermaid theme — 2 base vars */ --bg: #EDF5F3 /* slide background */ --fg: #194544 /* deep teal foreground */ /* Derived via color-mix() */ Node fill: color-mix(in srgb, var(--fg) 8%, var(--bg)) Node stroke: color-mix(in srgb, var(--fg) 25%, var(--bg)) Edge lines: color-mix(in srgb, var(--fg) 50%, var(--bg)) Labels: color-mix(in srgb, var(--fg) 40%, var(--bg)) Arrows: color-mix(in srgb, var(--fg) 85%, var(--bg)) /* Accent nodes use brand-teal directly */ ```mermaid graph LR A[User SOL] -->|Deposit| B[Marinade Pool] B -->|Mint| C[mSOL Token] style A fill:#EDF5F3,stroke:#447579 style B fill:#308D8A,stroke:#194544,color:#FFF style C fill:#94C9C8,stroke:#2B8784 ```
